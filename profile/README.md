PZ Wiki Modding is an organization with the goal of providing resources related to modding for the official [Project Zomboid wiki](https://pzwiki.net/wiki/Project_Zomboid_Wiki). It notably serves as the heart for the [PZ API Docs](https://pz-wiki-modding.github.io/PZ-API-Docs/index.html).

Two datasets are provided:
- [pz-scripts-data](https://github.com/PZ-Wiki-Modding/pz-scripts-data)
- [pz-translation-data](https://github.com/PZ-Wiki-Modding/pz-translation-data)

Two parsers are provided:
- [pz-lua-parser](https://github.com/PZ-Wiki-Modding/pz-lua-parser)
- [pz-java-parser](https://github.com/PZ-Wiki-Modding/pz-java-parser)

Two archives are available:
- [Archive.Project-Zomboid-Modding](https://github.com/PZ-Wiki-Modding/Archive.Project-Zomboid-Modding)
- [Archive.Azakaela-s-Modding-Tools](https://github.com/PZ-Wiki-Modding/Archive.Azakaela-s-Modding-Tools)

Two related mods to the Wiki:
- [Wiki That](https://github.com/PZ-Wiki-Modding/Wiki-That) - in-game ability to access wiki pages.
- [Wiki Tools](https://github.com/PZ-Wiki-Modding/Wiki-Tools) - in-game tools for the wiki editors.

A tool for players:
- [PZ Backup](https://github.com/PZ-Wiki-Modding/PZ-Backup) - easy backup of the current game installation to create a fully independent instance of the game to not be impacted by updates.
